aggregationStatus

  • aggregationStatus(request: any, response: any): Promise<void>
    • Defined in logic/federated/handlers.ts:451

    Request handler called when a client sends a POST request asking for the averaged model weights stored on server while training a task. The request is made for a given task and round. The request succeeds once CLIENTS_THRESHOLD % of clients sent their individual weights to the server for the given task and round. Every MODEL_SAVE_TIMESTEP rounds into the task, the requested averaged weights are saved under a JSON file at milestones/. The request's body must contain:

    • the client's ID
    • a timestamp corresponding to the time at which the request was made
